# PLAN MAESTRO

## IsiVoltPro Refrigeración y Climatización

### Superapp técnica para instalaciones, mantenimiento y servicio de campo

**Versión conceptual:** 1.0
**Marca principal:** IsiVoltPro
**Módulo:** Refrigeración y Climatización

---

# 1. VISIÓN DEL PRODUCTO

IsiVoltPro Refrigeración y Climatización será una aplicación profesional, móvil y offline orientada a:

* Técnicos frigoristas.
* Instaladores de climatización.
* Personal de mantenimiento.
* Empresas SAT.
* Responsables de instalaciones.
* Estudiantes y profesionales en formación.

La aplicación reunirá en una única plataforma:

1. Cálculos técnicos.
2. Información de refrigerantes.
3. Diagnóstico guiado.
4. Gestión de clientes y equipos.
5. Intervenciones de campo.
6. Fotografías y mediciones.
7. Informes PDF.
8. Inventario y control de refrigerante.
9. Mantenimiento preventivo.
10. Biblioteca técnica y formación.

La aplicación no será solamente una calculadora. Debe cubrir el flujo completo:

**Cliente → Instalación → Equipo → Intervención → Mediciones → Diagnóstico → Trabajo realizado → Materiales → Refrigerante → Fotografías → Firma → Informe PDF → Historial**
