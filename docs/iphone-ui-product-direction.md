# Dirección oficial de producto: iPhone UI

Este documento resume la dirección confirmada para la siguiente evolución de IsiVoltPro Refrigeración y Climatización.

## Principio de producto

La aplicación no debe convertirse en una colección genérica de calculadoras. Debe ser una superapp técnica de campo que acompañe el flujo completo:

Cliente -> Instalación -> Equipo -> Intervención -> Mediciones -> Diagnóstico -> Trabajo realizado -> Materiales -> Refrigerante -> Fotografías -> Firma -> Informe PDF -> Historial.

## Navegación principal

La barra inferior tiene cinco apartados:

- Inicio: accesos rápidos, nueva intervención, favoritos, actividad reciente y avisos.
- Herramientas: cálculos, diagnósticos y utilidades técnicas por categorías.
- Trabajo: clientes, instalaciones, equipos, intervenciones, fotos, historial e informes PDF.
- Biblioteca: refrigerantes, procedimientos, normativa, productos, repuestos y formación.
- Ajustes: perfil técnico, unidades, tema, offline y copias de seguridad.

Informes no es una categoría principal dentro de Herramientas. Vive dentro de Trabajo, aunque puede tener accesos rápidos desde Inicio o desde una intervención.

## Pantalla Inicio

Debe funcionar como panel de control para iPhone:

- Cabecera con marca IsiVoltPro y módulo Refrigeración y Climatización.
- Estado de conexión y guardado local.
- Nueva intervención como acción principal.
- Continuar última intervención.
- Escanear QR.
- Regla P/T.
- Crear cliente o equipo.
- Herramientas favoritas configurables.
- Actividad reciente y avisos.

## Herramientas

Categorías oficiales:

- Refrigerantes.
- Climatización.
- Conductos.
- Aerotermia / Hidráulica.
- Electricidad.
- Diagnóstico.

Cada herramienta debe tener:

1. Modo rápido para calcular directamente.
2. Modo explicado con fórmula, procedimiento, interpretación y errores frecuentes.
3. Resultado principal grande y legible.
4. Advertencias técnicas y nivel orientativo.
5. Fuente o referencia de datos cuando corresponda.
6. Botón Guardar en intervención.
7. Historial de cálculos recientes.

## Orden de implementación

### Etapa 1: iPhone UI navegable

- Sistema visual premium reutilizable.
- Navegación inferior con cinco apartados.
- Inicio con accesos directos grandes.
- Herramientas por categorías.
- Trabajo, Biblioteca y Ajustes navegables.
- Tema claro y oscuro.
- Responsive para móvil, tablet y escritorio.

### Etapa 2: herramientas frigoríficas existentes

- Regla P/T.
- Recalentamiento.
- Subenfriamiento.
- Vacío.
- Carga adicional.
- Conversor.
- Diagnóstico.

### Etapa 3: herramientas nuevas

- Psicrometría.
- Caudales y conductos.
- Pérdidas de carga.
- Carga térmica orientativa.
- Aerotermia e hidráulica.
- Electricidad asociada.
- UTA, fan coils e inductores.

## Reglas no negociables

- Conservar cálculos validados.
- Conservar base de refrigerantes trazable.
- Conservar almacenamiento offline e intervenciones actuales.
- No inventar datos técnicos.
- No recomendar mezclar refrigerantes.
- No recomendar carga solo por presión.
- Diferenciar presión absoluta y manométrica.
- Mostrar unidades siempre.
- Registrar fuente y fecha de datos.
- Indicar cuándo un resultado es orientativo.
