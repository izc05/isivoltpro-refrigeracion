# Base Codex refrigerante

La base inicial preparada por Codex se ha subido a la rama:

`codex/refrigerante-base`

Esa rama contiene una app Vite + React + TypeScript independiente con:

- Pantalla inicial de IsiVoltPro Refrigeracion y Climatizacion.
- Tarjetas de herramientas tecnicas.
- Biblioteca inicial de refrigerantes.
- Reglas tecnicas de seguridad visibles.
- `docs/plan-maestro.md` con el alcance completo del producto.

La rama no se fusiona directamente porque el repositorio principal ya contiene una aplicacion mas avanzada y publicada en GitHub Pages. Conviene revisar manualmente que partes rescatar antes de sustituir pantallas o documentacion existentes.
