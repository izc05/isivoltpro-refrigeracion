# Política de Privacidad

IsiVoltPro Refrigeración funciona localmente. No crea cuentas, no usa Firebase, no incluye publicidad, no recopila analítica y no envía datos personales a servidores externos.

Los datos de clientes, equipos, intervenciones, fotografías, firmas y ajustes se guardan en el dispositivo mediante IndexedDB/WebView. El usuario puede exportar o eliminar esos datos desde la aplicación.

La aplicación no solicita permisos de ubicación, contactos ni identificador publicitario. Las fotografías solo se añaden cuando el usuario selecciona o captura una imagen.
