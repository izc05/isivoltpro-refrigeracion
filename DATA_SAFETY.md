# Data Safety

- Recopilación: no se recopilan datos por el desarrollador.
- Compartición: no se comparten datos con terceros.
- Datos locales: intervenciones, clientes, equipos, fotografías y ajustes permanecen en el dispositivo.
- Seguridad: no hay transmisión de datos en V1.
- Eliminación: el usuario puede borrar datos locales desde ajustes o desinstalando la app.
